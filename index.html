Ants Admin

Admin Panel mudule for Rails 4 application

Installation

Add this line to your application's Gemfile:

```gem 'ants_admin', git: "https://github.com/ThanhKhoaIT/ants_admin.git"```

And then execute:

```$ bundle```

Vendors

  1. Store jStorage (http://www.jstorage.info)

  2. Popup bPopup (http://dinbror.dk/bpopup)

  3. Table Dynatable (http://www.dynatable.com)

Usage

Icons

Config model

Apply smart model: include AntsAdmin::SmartModel

    1. Disable for create page CREATE_DISABLED = true
    1. Set title page for model TITLE = "Item management"
    1. Set title for table TABLE_SHOW = "title", "description", "category", "show"
    1. Select attributes for search on model SEARCH_FOR = "title", "description"
    1. Config for action in table ACTIONS_LINK = "edit", "remove"
